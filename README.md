# hb-sdk 

> 本SDK为[huobi](https://www.huobi.pro/zh-cn)的[官方API](https://github.com/huobiapi/API_Docs/wiki/REST_api_reference)的ts封装。

### Install

```bash
npm install hb-sdk
```


### Usage

```bash
npm install hb-sdk
```
